QUESTION 01
What is an advantage of using an IPS?
An IPS can stop trigger packets. However, because they are installed inline they add some latency and jitter to the traffic. IDS sensors are passive and cannot stop trigger packets.
QUESTION 02
What is a characteristic of an IDS?
An IDS often requires assistance from other networking devices, such as routers and firewalls, to respond to an attack.
QUESTION 03
Which tool can perform real-time traffic and port analysis, and can also detect port scans, fingerprinting and buffer overflow attacks?
Snort is an open source IPS capable of real-time traffic and port analysis, packet logging, content searching/matching, and detecting probes, attacks, port scans, fingerprinting, and buffer overflow attacks.
QUESTION 04
Which Snort IPS feature enables a router to download rule sets directly from cisco.com or snort.org?
The Snort rule set pull feature allows a router to download rule sets directly from cisco.com or snort.org using one-time commands or periodic automated updates.
QUESTION 05
What is a minimum system requirement to activate Snort IPS functionality on a Cisco router?
Requirements to run Snort IPS: ISR 4300 or higher, K9 license, 8 GB RAM, and 8 GB flash.
QUESTION 06
What is PulledPork?
PulledPork is a rule management application for automatically downloading Snort rule updates. It requires an authorization code (oinkcode) from a snort.org account.
QUESTION 07
Which IPS signature trigger category uses a decoy server to divert attacks away from production devices?
Honey pot-based detection uses a decoy server to attract and divert attacks away from production devices, giving administrators time to analyze incoming attacks.
QUESTION 08
What situation will generate a true negative IPS alarm type?
True negative: benign normal traffic is correctly being ignored and forwarded without generating an alert.
QUESTION 09
What is provided by the fail open and close functionality of Snort IPS?
The Snort IPS fail open and close functionality can be configured to block traffic flow or bypass IPS checking in the event of an IPS engine failure.
QUESTION 10
What is a characteristic of the Community Rule Set type of Snort term-based subscriptions?
Community Rule Set is available for free, provides limited coverage, with 30-day delayed signature access and no Cisco support. Subscriber Rule Set is paid and uses Cisco Talos research.
QUESTION 11
What is a characteristic of the connectivity policy setting when configuring Snort threat protection?
Connectivity is the least secure option. Balanced is the mid-range option. Security is the most secure option.
QUESTION 12
What is contained in an OVA file?
An OVA (Open Virtualization Archive) file contains a compressed, installable version of a virtual machine.
QUESTION 13
What is a network tap?
A network tap is a passive splitting device implemented inline on the network that forwards all traffic, including physical layer errors, to an analysis device.
QUESTION 14
Which statement describes the function of the SPAN tool used in a Cisco switch?
SPAN (Switched Port Analyzer) mirrors traffic from one port to another port on the same switch where a network analyzer is connected.
QUESTION 15
A network administrator is trying to download a valid file from an internal server. However, the process triggers an alert on a NMS tool. What condition describes this alert?
A false positive is when an alert is generated for benign activity — normal traffic that should not trigger an alarm but does.
QUESTION 16
What is an advantage of HIPS that is not provided by IDS?
HIPS monitors and protects OS and critical system processes specific to that host. It can be thought of as a combination of antivirus, antimalware, and a firewall.
QUESTION 17
What information must an IPS track in order to detect attacks matching a composite signature?
A composite (stateful) signature identifies a sequence of operations across multiple hosts over time. The IPS sensor must maintain state information for the specific attack.
Layer 2 & Endpoint Security Modules 13–14

QUESTION 01
Why are traditional network security perimeters not suitable for the latest consumer-based network endpoint devices?
Traditional security is not suited for new network devices that are mobile, frequently access cloud storage, and may be personal devices.
QUESTION 02
In an 802.1x deployment, which device is a supplicant?
In 802.1x, a supplicant is the end-user device (such as a laptop) attempting to attach to the WLAN.
QUESTION 03
A company implements 802.1X security on the corporate network. A PC is attached to the network but has not authenticated yet. Which 802.1X state is associated with this PC?
When a port is configured for 802.1X, the port starts in the unauthorized state and stays that way until the client has successfully authenticated.
QUESTION 04
Which command is used as part of the 802.1X configuration to designate the authentication method that will be used?
The aaa authentication dot1x default group radius command specifies RADIUS as the method for 802.1X port-based authentication.
QUESTION 05
What is involved in an IP address spoofing attack?
In an IP address spoofing attack, the IP address of a legitimate network host is hijacked and used by a rogue node to pose as a valid node on the network.
QUESTION 06
At which layer of the OSI model does Spanning Tree Protocol operate?
STP is a Layer 2 technology for preventing Layer 2 loops between redundant switch paths.
QUESTION 07
A network administrator uses the spanning-tree loopguard default global configuration command to enable Loop Guard on switches. What components in a LAN are protected with Loop Guard?
The spanning-tree loopguard default command enables Loop Guard on all point-to-point links.
QUESTION 08
Which procedure is recommended to mitigate the chances of ARP spoofing?
To mitigate ARP spoofing: enable DHCP snooping globally and on selected VLANs, enable DAI on selected VLANs, and configure trusted interfaces for DHCP snooping and ARP inspection.
QUESTION 09
Which protocol should be used to mitigate the vulnerability of using Telnet to remotely manage network devices?
SSH encrypts data communications between two network devices. Telnet uses plain text and credentials can be captured if traffic is intercepted.
QUESTION 10
How can DHCP spoofing attacks be mitigated?
DHCP spoofing attacks can be mitigated by using DHCP snooping on trusted ports.
QUESTION 11
What is the behavior of a switch as a result of a successful CAM table attack?
When the CAM table is full, no new MAC addresses can be added and the switch forwards all received frames to all other ports, allowing an attacker to capture all flooded traffic.
QUESTION 12
Which protocol defines port-based authentication to restrict unauthorized hosts from connecting to the LAN through publicly accessible switch ports?
802.1x is an IEEE standard that defines port-based access control, providing protection from unauthorized clients by authenticating each client attempting to connect to the LAN.
QUESTION 13
What device is considered a supplicant during the 802.1X authentication process?
The supplicant is the client requesting network access; the authenticator is the switch controlling physical access; the authentication server performs actual authentication.
QUESTION 14
Which term describes the role of a Cisco switch in the 802.1X port-based access control?
The switch acts as the authenticator — it controls physical access to the network based on the authentication status of the client.
QUESTION 15
What type of data does the DLP feature of Cisco Email Security Appliance scan in order to prevent customer data from being leaked outside of the company?
Cisco ESAs control outbound messages through DLP to ensure outbound messages comply with industry standards and are protected in transit.
QUESTION 16
What is the goal of the Cisco NAC framework and the Cisco NAC appliance?
NAC ensures wired and wireless endpoints that want to gain access to the network adhere to the requirements defined by the security policy.
QUESTION 17
Which Cisco solution helps prevent MAC and IP address spoofing attacks?
IP Source Guard (IPSG) prevents MAC and IP address spoofing. DAI prevents ARP spoofing. DHCP Snooping prevents DHCP starvation. Port Security prevents MAC table overflow attacks.
QUESTION 18
What Layer 2 attack is mitigated by disabling Dynamic Trunking Protocol?
VLAN hopping attacks can be mitigated by disabling DTP and setting the native VLAN of trunk links to VLANs not in use.
QUESTION 19
What is the result of a DHCP starvation attack?
DHCP starvation attacks send many DHCPDISCOVER messages to lease the entire pool of available IP addresses, denying them to legitimate hosts (DoS).
QUESTION 20
A network administrator is configuring DAI on a switch with the command ip arp inspection validate dst-mac. What is the purpose of this command?
Destination MAC validation checks the destination MAC in the Ethernet header against the target MAC address in the ARP body.

QUESTION 01
Which algorithm can ensure data integrity?
Data integrity is ensured by implementing SHA-2, SHA-3, or MD5 hash algorithms. The MD5 message digest algorithm is still widely in use.
QUESTION 02
What is the keyspace of an encryption algorithm?
The keyspace of an encryption algorithm is the set of all possible key values. Keys with n bits produce a keyspace with 2ⁿ possible key values.
QUESTION 03
Alice and Bob are using a digital signature to sign a document. What key should Alice use to sign the document so that Bob can make sure that the document came from Alice?
Alice uses her private key to encrypt the message digest. The message, encrypted message digest, and public key are used to create the signed document.
QUESTION 04
What is another name for confidentiality of information?
Privacy is another name for confidentiality. Accuracy, consistency, and trustworthiness describe data integrity.
QUESTION 05
As data is being stored on a local hard disk, which method would secure the data from unauthorized access?
Data encryption converts data into a form where only a trusted, authorized person with a secret key or password can decrypt and access the original form.
QUESTION 06
What popular encryption algorithm requires that both the sender and receiver know a pre-shared key?
AES (Advanced Encryption Standard) is a popular symmetric encryption algorithm where each communicating party needs to know the pre-shared key.
QUESTION 07
What technology supports asymmetric key encryption used in IPsec VPNs?
IKE (Internet Key Exchange) is a protocol supporting asymmetric encryption algorithms. It is used to securely exchange encryption keys in IPsec VPN setup.
QUESTION 08
Which statement describes asymmetric encryption algorithms?
DES, 3DES, and AES are symmetric algorithms. Asymmetric algorithms are relatively slow because they are based on difficult computational algorithms.
QUESTION 09
Two users must authenticate each other using digital certificates and a CA. Which option describes the CA authentication procedure?
Both users must obtain their own digital certificate from a CA. They submit a request, CA performs technical verification (out-of-band), and the user retrieves the certificate in-band.
QUESTION 10
The message "fghgpf vjg ecuvng" was encrypted using a Caesar cipher with a key of 2. What is the plaintext message?
With key = 2, each letter is shifted back 2 places: f=d, g=e, h=f, p=n, v=t, j=h, e=c, c=a, u=s, n=l → "defend the castle".
QUESTION 11
In a hierarchical CA topology, where can a subordinate CA obtain a certificate for itself?
In a hierarchical CA topology, a subordinate CA can obtain its certificate from the root CA or another subordinate CA at a higher level. Only the root CA can issue a self-signing certificate.
QUESTION 12
What is the purpose for using digital signatures for code signing?
Code signing is used to verify the integrity of executable files downloaded from a vendor website using digital certificates.
QUESTION 13
What technology has a function of using trusted third-party protocols to issue credentials that are accepted as an authoritative identity?
A PKI Certificate Authority is a trusted third-party entity that issues PKI certificates, providing authenticity, confidentiality, integrity, and nonrepudiation services.
QUESTION 14
Which requirement of secure communications is ensured by the implementation of MD5 or SHA hash generating algorithms?
Integrity is ensured by implementing MD5 or SHA hash algorithms. Authentication uses HMAC protocols. Confidentiality uses DES, 3DES, AES, RSA, or PKI.
QUESTION 15
What is an example of the one-time pad cipher?
RC4 is an example of the one-time pad cipher widely used on the Internet. Caesar is a substitution cipher; Vigenère is based on Caesar; rail fence is a transposition cipher.
QUESTION 16
A company is developing a security policy. A hash value should only be recalculated with a predetermined code, thus ensuring the validity of data source. Which aspect of secure communications is addressed?
Origin authentication guarantees that the message is not a forgery and does actually come from who it states it comes from.
QUESTION 17
What is the purpose of a digital certificate?
Digital signatures commonly use digital certificates to verify the identity of the originator, authenticate a vendor website, and establish an encrypted connection to exchange confidential data.

QUESTION 01
What technology is used to negotiate security associations and calculate shared keys for an IPsec VPN tunnel?
IKE (Internet Key Exchange) negotiates security associations (SAs) and calculates shared keys for IPsec VPN tunnels.
QUESTION 02
What takes place during IKE Phase 2 when establishing an IPsec VPN?
During IKE Phase 2, IPsec peers exchange the IPsec security associations (SAs) that each peer is willing to use to establish the IPsec tunnel.
QUESTION 03
When the CLI is used to configure an ISR for a site-to-site VPN connection, what is the purpose of the crypto map command in interface configuration mode?
The crypto map command, along with the policy name, is used to bind the interface to the ISAKMP policy created previously.
QUESTION 04
Router R1 has configured ISAKMP policies numbered 1, 5, 9, and 203. Router R2 only has default policies. How will R1 attempt to negotiate the IKE Phase 1 ISAKMP tunnel with R2?
Peers attempt to negotiate using the policy with the lowest number (highest priority). Peers do not require matching priority numbers.
QUESTION 05
Which statement describes the effect of key length in deterring an attacker from hacking through an encryption key?
The longer the key, the harder it is to break. A 64-bit key may take 1 year; a 128-bit key may take 10¹⁹ years to decrypt with a sophisticated computer.
QUESTION 06
Which protocol creates a virtual point-to-point connection to tunnel unencrypted traffic between Cisco routers from a variety of protocols?
GRE (Generic Routing Encapsulation) encapsulates multiprotocol traffic between Cisco routers. GRE does not encrypt data.
QUESTION 07
How is "tunneling" accomplished in a VPN?
Tunneling encapsulates packets with headers from one or more VPN protocols before they are sent across the third party network.
QUESTION 08
Which statement accurately describes a characteristic of IPsec?
IPsec is a framework of open standards that relies on existing algorithms for confidentiality, integrity, authentication, and secure key exchange.
QUESTION 09
Which is a requirement of a site-to-site VPN?
Site-to-site VPNs are static and require a VPN gateway at each end to encapsulate and decapsulate traffic. Hosts have no knowledge of the VPN.
QUESTION 10
Consider the following configuration on a Cisco ASA: crypto ipsec transform-set ESP-DES-SHA esp-des esp-sha-hmac. What is the purpose of this command?
The transform set defines what encryption and authentication schemes can be used, negotiated during Phase 2 of the IPsec VPN connection process.
QUESTION 11
What is needed to define interesting traffic in the creation of an IPsec tunnel?
An access list with a permit statement identifies interesting traffic. Once matched, tunnel security associations can be negotiated.
QUESTION 12
What is a function of the GRE protocol?
GRE supports multiprotocol tunneling and can encapsulate multiple OSI Layer 3 protocol packet types inside an IP tunnel. GRE does not provide encryption.
QUESTION 13
Two corporations have just completed a merger. The network engineer has been asked to connect the two corporate networks without the expense of leased lines. Which solution would be the most cost effective method of providing a proper and secure connection between the two corporate networks?
Site-to-site VPN is an extension of a classic WAN network that provides a static interconnection of entire networks — the most cost-effective method for connecting two corporate networks.
QUESTION 14
What type of traffic is supported by IPsec?
IPsec only supports unicast traffic. If multicast traffic needs to travel through a tunnel, a GRE tunnel must be configured between the peers.

QUESTION 01
A network analyst wants to monitor the activity of all new interns. Which type of security testing would track when the interns sign on and sign off the network?
An integrity checking system can report login and logout activities. Network scanning detects user names, groups, and shared resources by scanning TCP ports.
QUESTION 02
What testing tool is available for network administrators who need a GUI version of Nmap?
Zenmap is the GUI version of Nmap. Nessus scans for software vulnerabilities. SuperScan is a Microsoft port scanning tool. SIEM is used for real-time security event reporting.
QUESTION 03
What is the goal of network penetration testing?
Penetration testing determines the possible consequences of successful attacks on the network. Vulnerability scanning detects weaknesses. Password cracking detects weak passwords.
QUESTION 04
How does network scanning help assess operations security?
Network scanning helps identify open TCP and UDP ports that could be targets of an attack, strengthening network security.
QUESTION 05
What can be configured as part of a network object?
Network objects can be configured with an IP address and mask. Service objects can be configured with a protocol or port ranges.
QUESTION 06
What is the function of a policy map configuration when an ASA firewall is being configured?
Policy maps bind class maps with actions. Class maps identify Layer 3/4 traffic. Service policies attach the policy map to an interface.
QUESTION 07
What is the purpose of configuring an IP address on an ASA device in transparent mode?
An ASA in transparent mode functions like a Layer 2 device and does not support dynamic routing protocols, VPNs, QoS, or DHCP. The IP address is used for management.
QUESTION 08
Which license provides up to 50 IPsec VPN users on an ASA 5506-X device?
The Security Plus upgrade license supports a higher connection capacity and up to 50 IPsec VPN users on the ASA 5506-X.
QUESTION 09
What mechanism is used by an ASA device to allow inspected outbound traffic to return to the originating sender who is on an inside network?
Stateful packet inspection allows return traffic sourced on the outside network to be received by the originating sender on the internal network.
QUESTION 10
What interface configuration command is used on an ASA to request an IP address from an upstream DSL device?
PPPoE is used when connecting to an upstream DSL device. The ip address pppoe command is used for point-to-point connectivity over Ethernet services.
QUESTION 11
What is the purpose of the Tripwire network testing tool?
Tripwire assesses configuration against established policies, recommended best practices, and compliance standards.
QUESTION 12
A network analyst is testing the security of the systems and networks of a corporation. What tool could be used to audit and recover passwords?
L0phtCrack is a password auditing and recovery application. SuperScan detects open ports. Nessus scans vulnerabilities. Metasploit aids penetration testing.
